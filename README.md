# [jemdoc](https://jemdoc.jaboc.net/)

My homepage hosted by [jemdoc](https://jemdoc.jaboc.net/).

## Usage

Run `./build.sh` on Unix-like shells, or `./build.ps1` in PowerShell, after modifying any `*.jemdoc` files.

Both scripts rebuild pages when any source file, `MENU`, `jemdoc.py`, or `jemdoc.css` changes.
If a repo-local Python 2 runtime is available under `.python2/`, it will be used automatically.
