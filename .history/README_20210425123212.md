# [jemdoc](https://jemdoc.jaboc.net/)

My homepage hosted by [jemdoc](https://jemdoc.jaboc.net/).

## Usage
First, `jemdoc.py` all `.jemdoc` files like this:
> python2 jemdoc.py index.jemdoc
Then `git push` as usual
